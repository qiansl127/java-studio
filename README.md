## Usage

### Option 1
Call main method in class `com.frank.java.studio.PlayGround` in your IDE.

### Option 2
Use the Gradle plugin in your IntelliJ, and choose the jar step.

```bash
java -jar ./build/libs/java-studio-1.0-SNAPSHOT.jar
```
